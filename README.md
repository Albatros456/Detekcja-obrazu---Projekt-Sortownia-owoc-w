# Detekcja-obrazu---Projekt-Sortownia-owoc-w
Celem projektu było zaprojektowanie i implementacja systemu automatycznej detekcji oraz klasyfikacji owoców w czasie rzeczywistym za pomocą kamery cyfrowej. Do realizacji zadania zastosowano konwolucyjne sieci neuronowe modelu YOLOv8 (ang. You Only Look Once) w wersji YOLOv8n.
